## Creating a Python virtual environment
`py -m venv venv`
## Activate environment
`./venv/Scripts/activate`

## installing packages
`pip3 install -r requirements.txt`

## Start
* ./src/main.py

**or**

* ./src/main.py --reload

# Restful docs
* [http://localhost:8000/docs](http://localhost:8000/docs)
